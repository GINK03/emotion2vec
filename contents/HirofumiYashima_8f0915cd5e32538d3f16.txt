The object contains the following slots:\n\nmain\nLotka-Volterra equations for predator and prey.\n\nparms\nVector with the named parameters of the model:\n\nk1\ngrowth rate of the prey population,\n\nk2\nencounter rate of predator and prey,\n\nk3\ndeath rate of the predator population.\n\ntimes\nSimulation time and integration interval.\n\ninit\nVector with start values for predator and prey.\n\nDetails\n\nTo see all details, please have a look into the implementation.\n\nReferences\n\nLotka, A. new(\"odeModel\",\n  main = function (time, init, parms) {\n    x <- init\n    p <- parms\n    dx1 <-   p[\"k1\"] * x[1] - p[\"k2\"] * x[1] * x[2]\n    dx2 <- - p[\"k3\"] * x[2] + p[\"k2\"] * x[1] * x[2]\n    list(c(dx1, dx2))\n  },\n  parms  = c(k1=0.2, k2=0.2, k3=0.2),\n  times  = c(from=0, to=100, by=0.5),\n  init   = c(prey=0.5, predator=1),\n  solver = \"rk4\"\n)\n\n\n\nexample(lv)
