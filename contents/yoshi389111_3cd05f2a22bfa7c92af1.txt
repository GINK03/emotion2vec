Description of the option. (\u30aa\u30d7\u30b7\u30e7\u30f3\u306e\u8aac\u660e)\n\n> The normal  mode does not allow random  lossage; therefore the system is not allowed  to crash,  mung user files, etc.  If the server wants to provide  random lossage, it must first ask for permission from the user by sending IAC WILL RANDOMLY-LOSE.\n\n\u666e\u901a\u306e\u30e2\u30fc\u30c9\u3067\u306f\uff0c\u30e9\u30f3\u30c0\u30e0\u55aa\u5931\u6a5f\u80fd\u306f\u8a31\u3055\u308c\u306a\u3044\uff1b\u305d\u308c\u3086\u3048\u306b\uff0c\u305d\u306e\u30b7\u30b9\u30c6\u30e0\u306f\u30af\u30e9\u30c3\u30b7\u30e5\u3059\u308b\u3053\u3068\uff0c\u3046\u3063\u304b\u308a(\u3042\u308b\u3044\u306f\uff0c\u60aa\u610f\u3092\u6301\u3063\u3066)\u30e6\u30fc\u30b6\u30d5\u30a1\u30a4\u30eb\u3092\u58ca\u3059\u3053\u3068\u7b49\u306f\u8a31\u3055\u308c\u306a\u3044\uff0e\u3082\u3057\u30b5\u30fc\u30d0\u304c\u30e9\u30f3\u30c0\u30e0\u55aa\u5931\u6a5f\u80fd\u3092\u63d0\u4f9b\u3057\u305f\u3044\u5834\u5408\u306b\u306f\uff0c\u59cb\u3081\u306b IAC WILL RANDOMLY-LOSE \u30b3\u30de\u30f3\u30c9\u3092\u9001\u3063\u3066\uff0c\u30e6\u30fc\u30b6\u306b\u8a31\u53ef\u3092\u6c42\u3081\u306a\u304f\u3066\u306f\u306a\u3089\u306a\u3044\uff0e\n\n> If the user wants  to permit  the server  to randomly lose, it replys with  IAC  DO   RANDOMLY-LOSE.
